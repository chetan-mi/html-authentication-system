<h1>HTML Authentication System</h1>

<h2>Project Overview</h2>
<p>
This project implements a structured static authentication flow using pure HTML.
</p>
<p>
The system simulates a real-world authentication process including login,
registration, password recovery, password reset, and dashboard navigation.
</p>
<p>
The entire application is built without CSS, JavaScript, or backend logic,
strictly following assignment requirements.
</p>

<hr>

<h2>Features Implemented</h2>
<ul>
    <li>Login Page</li>
    <li>Registration Page</li>
    <li>Forgot Password Page</li>
    <li>Reset Password Page</li>
    <li>Dashboard Page</li>
    <li>Proper inter-page navigation using &lt;a href&gt;</li>
    <li>Form-based redirection using the action attribute</li>
    <li>Clean semantic HTML structure</li>
</ul>

<hr>

<h2>Pages Included</h2>
<ol>
    <li>login.html</li>
    <li>register.html</li>
    <li>forgot-password.html</li>
    <li>reset-password.html</li>
    <li>dashboard.html</li>
</ol>

<hr>

<h2>Navigation Flow</h2>
<ul>
    <li>Login → Dashboard</li>
    <li>Dashboard → Logout → Login</li>
    <li>Login → Register</li>
    <li>Register → Login</li>
    <li>Login → Forgot Password</li>
    <li>Forgot Password → Login</li>
    <li>Reset Password → Login</li>
</ul>

<hr>

<h2>Technical Implementation</h2>
<ul>
    <li>Built using HTML5</li>
    <li>No CSS styling (as per assignment rules)</li>
    <li>No JavaScript validation</li>
    <li>No backend integration</li>
    <li>Static form submission and hyperlink navigation only</li>
</ul>

<hr>

<h2>Purpose of the Project</h2>
<p>
This project demonstrates understanding of HTML forms,
page structuring, navigation architecture,
and static authentication flow modeling.
</p>

<p>
It serves as a foundational web development exercise
focusing on structural clarity and logical page flow.
</p>
